# 2026-02-10 — UI 重构：Modern B-End / Enterprise SaaS 风格

## 概述

根据最新需求，对 UI 进行了深度重构，目标是打造类似飞书/钉钉/企查查的 **Modern B-End** 风格。
核心关键词：**扁平 (Flat)**、**边框导向 (Border-based)**、**高密度 (High Density)**、**无阴影 (No Shadow)**。

---

## 主要变更 ✅

### 1. 全局样式 (Global Styles)
- 🎨 **配色更新**：主色调调整为 **Feishu Blue (#3370FF)**。
- 🚫 **去除阴影**：移除 `.card`, `.btn` 等组件的 `box-shadow`，改为 `border` 导向。
- 📏 **圆角调整**：统一使用更小的圆角（6px/8px），体现专业感。
- 🖼 **背景色**：统一使用 `#F5F6F7` 灰色背景，卡片使用纯白 `#FFFFFF`。

### 2. 企业详情页 (Enterprise Detail)
- **重构目标**：对标企查查/天眼查，高密度信息展示。
- **头部设计**：
  - 纯白背景，底部边框。
  - 大号 Logo + 企业名称 + 状态标签（存续/在孵）。
  - 核心指标网格（法人、资金、日期、代码）直接展示。
  - 右侧操作按钮组（关联妙记、导出、背调）。
- **Sticky Tabs**：吸顶选项卡，方便长页面导航。
- **信息分层**：
  - **工商信息**：密集网格布局。
  - **走访记录**：时间轴/列表式卡片。
  - **风险扫描**：独立模块。

### 3. 首页 (Dashboard)
- **极简 B 端风格**：移除所有装饰性渐变和阴影。
- **核心指标**：扁平化卡片，使用边框区分，红色/绿色趋势文字。
- **场景入口**：列表式/密集卡片，强调数据（待办数、覆盖率）。
- **动态 Feed**：紧凑列表，使用图标颜色区分类型（Success/Warning/Info）。

### 4. 企业列表 (Enterprise List)
- **扁平化卡片**：移除悬浮阴影，改为悬浮边框变色 (`hover:border-blue-400`)。
- **信息密度提升**：在卡片内展示更多关键字段（法人、成立日期、地址）。
- **视觉降噪**：减少色块面积，使用文字和边框传达层级。

### 5. 其他页面 (Visit / Policy / Incubator)
- **统一头部**：使用白色背景 + 底部边框的 B 端标准头部。
- **一致性**：所有页面应用相同的卡片风格和背景色。

---

## 视觉规范 (Design System)

| 维度 | 旧风格 (Warm/Shadow) | 新风格 (Modern B-End) |
|------|---------------------|----------------------|
| **阴影** | `shadow-sm`, `shadow-md` | **None** (仅用 Border) |
| **边框** | 浅色 `#E2E8F0` | 浅色 `#E2E8F0`，Hover 变蓝 |
| **圆角** | 8px - 12px | **6px - 8px** (更锐利) |
| **主色** | `#3B82F6` (Tailwind Blue) | **`#3370FF` (Feishu Blue)** |
| **背景** | 渐变/纯白 | **`#F5F6F7` (灰色背景)** |
| **布局** | 宽松 | **紧凑 (Compact)** |
